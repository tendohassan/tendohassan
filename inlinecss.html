CASCADING STYLE SHEETS (CSS)

Css is used to style and describe the layout of a web page.

Css can be added in a web page via three methods as listed below.

  1. Inline css
  2. External css
  3. Internal css

Css has some of the following functions in a web page.

  • Changing the color of an element.
  • Adjusting the size and dimensions of an element
  • Adjusting the layout of elements
